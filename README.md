# Messenger-DOED
Messenger DOED 2.0 Version

Инструкция для тех кто хочет запустить мессенджер локально:

1. Скачайте файл мессенджера

2. Переходиту в корневую папку

3. Выполните команду pip install -r requirements.txt

4. Выполните команду uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

5. Переходите в браузере на https://localhost:8000
